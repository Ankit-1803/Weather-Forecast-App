# Weather Forecast App

This is a simple weather application made using HTML, CSS and JavaScript.

The app allows users to search weather information by city name or use their current location to get weather details. Weather data is fetched from the OpenWeatherMap API.

## Features

* Search weather by city name
* Get weather using current location
* Display temperature, humidity, pressure and wind speed
* Weather icons based on current conditions
* Responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API
* Geolocation API
